# ASP.MVC_CustomerSystem

目的:當初想做做看簡易版的客戶資料管理系統


功能:  <br>
       • 登入依照SQL資料庫裡的帳號密碼做比對 <br>
	     • 客戶資料管理可以從資料庫查到客戶基本資料，並且可以幫客戶新增、刪除與修改資料 <br>
	   
未來展望: <br>
     • 功能在增加搜尋功能 <br>
		 • 美化版面 <br>
